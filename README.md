# TH Pagweb Intranet

Aplicación base de intranet construida con React + Vite.

## Módulos implementados

- Inicio (panel de indicadores)
- Noticias internas
- Documentos
- Usuarios

## Tecnologías

- React
- React Router DOM
- Vite
- ESLint

## Scripts

- `npm run dev`: inicia el entorno de desarrollo
- `npm run build`: genera build de producción
- `npm run preview`: sirve la build local
- `npm run lint`: valida reglas de código

## Ejecución local

1. Instalar dependencias:
   - `npm install`
2. Ejecutar app:
   - `npm run dev`
